Présentation du projet SPA avec API

Je voulais faire un site avec l'api d'un jeu auquel je joue et j'ai trouvé celui de
Clash of Clans. C'est un api avec ses propres méthodes alors je devais utiliser celles-ci
et non request. On accède en premier à une page d'accueil avec des données sur le clan.
On peut ensuite aller à la liste des membres du clan. On y retrouve le nom, le tag et
un boutton pour voir les détails de celui-ci. Quand on arrive dans la page de détails,
on retrouve une listes d'info sur le joueur. On peut revenir à la liste ou à l'accueil
avec les liens de navigation à gauche. 


Inititalisation du projet : npm init -y

Installation des paquets : npm i express dotenv path clash-of-clans-api request

Créations des modules : npm i

Démarrage du serveur : npm start / node server.js
